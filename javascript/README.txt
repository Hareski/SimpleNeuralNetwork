Comment ca marche:

1. lancer le main dans "../Neurone" avec "./prog 2 1 " ou "./prog 2 2 " au moins une fois et ctrl + c

2. dans le terminal, se placer dans le repertoire javascript (cd javascript)

3. lancer un serveur sur localhost... easy way : dans le terminal, entrer "python -m SimpleHTTPServer 8080"
 le '8080' peut etre modifiE ... c'est le port

4. allez dans votre navigateur favori et rush: "http://localhost:8080/?path=data"
si vous avez modifiE le port... vous savez quoi faire ici        ^ ^

on peut aussi modifier le path:
data = fichier correspondant au dernier test
ET = fichier ET
OU = fichier OU

5. le fichier "data" n'est pas chargE en meme temps ... donc vous aurez ptet a recharger la page plusieurs fois
jusqu'a ce que le File Status dise "<machin> - Done" machin = et ou ou
parfois ouvrir la console aide...jsp pq (F12)....parfois ctrl + f5

6. enjoy
